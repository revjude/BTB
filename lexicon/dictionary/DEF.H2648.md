**<big><span class="ashebrew">‎חָפַז‎</span></big>** **châphaz** <small>*(khaw-faz')*</small> ==**to hurry**==  <sup><small>[[TEHMC#H:V|H:V]]</small></sup> 
<small>a primitive root</small>

$\quad$ (make) haste (away), tremble.
- properly, ***to start up suddenly***, i.e. (by implication) ***to hasten away***, ***to fear***

**1)** to hurry, flee, hasten, fear, be terrified
$\quad$ **1a)** (Qal) to be in a hurry, be alarmed
$\quad$ **1b)** (Niphal) to be in a hurry
