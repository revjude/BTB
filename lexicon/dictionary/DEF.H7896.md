**<big><span class="ashebrew">‎שִׁית‎</span></big>** **shîyth** <small>*(sheeth)*</small> ==**to set: make**==  <sup><small>[[TEHMC#H:V|H:V]]</small></sup> 
<small>a primitive root</small>

$\quad$ apply, appoint, array, bring, consider, lay (up), let alone, [idiom] look, make, mark, put (on), [phrase] regard, set, shew, be stayed, [idiom] take.
- ***to place*** (in a very wide application)

**1)** to put, set
$\quad$ **1a)** (Qal)
$\quad$ $\quad$ **1a1)** to put, lay (hand upon)
$\quad$ $\quad$ **1a2)** to set, station, appoint, fix, set mind to
$\quad$ $\quad$ **1a3)** to constitute, make (one something), make like, perform
$\quad$ $\quad$ **1a4)** to take one's stand
$\quad$ $\quad$ **1a5)** to lay waste
$\quad$ **1b)** (Hophal) to be imposed, be set upon
