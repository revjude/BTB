**<big><span class="ashebrew">‎שׂוּם‎</span></big>** **sûwm** <small>*(soom)*</small> ==**to set: make**==  <sup><small>[[TEHMC#H:V|H:V]]</small></sup> 
<small>or **<big><span class="ashebrew">שִׂים</span></big>** **sîym** <small>*(seem)*</small>; a primitive root</small>

$\quad$ [idiom] any wise, appoint, bring, call (a name), care, cast in, change, charge, commit, consider, convey, determine, [phrase] disguise, dispose, do, get, give, heap up, hold, impute, lay (down, up), leave, look, make (out), mark, [phrase] name, [idiom] on, ordain, order, [phrase] paint, place, preserve, purpose, put (on), [phrase] regard, rehearse, reward, (cause to) set (on, up), shew, [phrase] stedfastly, take, [idiom] tell, [phrase] tread down, (over-)turn, [idiom] wholly, work.
- ***to put*** (used in a great variety of applications, literal, figurative, inferentially, and elliptically)

**1)** to put, place, set, appoint, make
$\quad$ **1a)** (Qal)
$\quad$ $\quad$ **1a1)** to put, set, lay, put or lay upon, lay (violent) hands on
$\quad$ $\quad$ **1a2)** to set, direct, direct toward
$\quad$ $\quad$ $\quad$ **1a2a)** to extend (compassion) (fig)
$\quad$ $\quad$ **1a3)** to set, ordain, establish, found, appoint, constitute, make, determine, fix
$\quad$ $\quad$ **1a4)** to set, station, put, set in place, plant, fix
$\quad$ $\quad$ **1a5)** to make, make for, transform into, constitute, fashion, work, bring to pass, appoint, give
$\quad$ **1b)** (Hiphil) to set or make for a sign
$\quad$ **1c)** (Hophal) to be set
