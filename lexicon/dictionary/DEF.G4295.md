**<big><span class="asgreek">‎πρόκειμαι‎</span></big>** **prókeimai** <small>*(prok'-i-mahee)*</small> ==**to set before**==  <sup><small>[[TEGMC#G:V|G:V]]</small></sup> 
<small>from **<big>[[G4253|‎πρό‎]]</big>** and **<big>[[G2749|‎κεῖμαι‎]]</big>**</small>

$\quad$ be first, set before (forth).
- to lie before the view, i.e. (figuratively) to be present (to the mind), to stand forth (as an example or reward)
