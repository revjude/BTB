**Hiph\`il** ***Perfect*** suffix <span class="bdbLangHebrew">‎הדיחנו‎</span> [[BDB#Kt|Kt]] [[Jer.51.34|Jer 51:34]] [[BDB#Qr|Qr]] <span class="bdbLangHebrew">‎הֱדִיחָ֑נִי‎</span> but read <span class="bdbLangHebrew">‎הִדִּיחני‎</span> from <span class="bdbLangHebrew">‎נדח‎</span> ([[BDB#Hi|Hi]], compare [[Jer.50.17|Jer 50:17]]); ***Imperfect*** 3 masculine singular <span class="bdbLangHebrew">‎יָדִיחַ‎</span> [[Isa.4.4|Isa 4:4]] 3 masculine plural <span class="bdbLangHebrew">‎יָדִיחוּ‎</span> [[2Chr.4.6|2Chr 4:6]] + [[Ezek.40.38|Ezek 40:38]] ([[BDB#Co|Co]] conjecture <span class="bdbLangHebrew">‎הַקָּדִים‎</span>); —
