 <span class="bdbLangHebrew">‎נֶ֫שֶׁף‎</span> **noun masculine** ==twilight== (properly ***twilight-breeze***, compare <span class="bdbLangHebrew">‎לְרוּחַ הַיּוֺם‎</span> [[Gen.3.8|Gen 3:8]] Late Hebrew id.; [[BDB#Tg|ᵑ7]] <span class="bdbLangHebrew">‎נְשֶׁף‎</span>, <span class="bdbLangHebrew">‎נִשְׁפָּא‎</span>; — <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎נ‎</span> absolute [[1Sam.30.17|1Sam 30:17]] +, <span class="bdbLangHebrew">‎נָ֑שֶׁף‎</span> [[Jer.13.16|Jer 13:16]] construct [[Isa.21.4|Isa 21:4]] suffix <span class="bdbLangHebrew">‎נִשְׁפּוֺ‎</span> [[Job.3.9|Job 3:9]] — 

**1.** ***evening twilight*** [[2Kgs.7.5|2Kgs 7:5]]; [[2Kgs.7.7|2Kgs 7:7]] (compare <span class="bdbLangHebrew">‎לַיְלָה‎</span> v[[2Kgs.7.12|2Kgs 7:12]]); opposed to <span class="bdbLangHebrew">‎בֹּקֶר‎</span> [[Isa.5.11|Isa 5:11]] time of concealment [[Job.24.15|Job 24:15]]; [[Prov.7.9|Prov 7:9]] (+ <span class="bdbLangHebrew">‎עֶרֶב יוֺם‎</span>, <span class="bdbLangHebrew">‎אִישׁוֺן לַיְלָה וַאֲפֵלָה‎</span>, to emphasize sin that shuns the day); of refreshing [[Isa.21.4|Isa 21:4]] <span class="bdbLangHebrew">‎חִשְׁקִיׅ‎</span> <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎(נ‎</span>; of stumbling, in dim light [[Isa.59.10|Isa 59:10]] (opposed to <span class="bdbLangHebrew">‎צָהֳרַיִם‎</span>), so <span class="bdbLangHebrew">‎הָרֵי נָ֑שֶׁף‎</span> [[Jer.13.16|Jer 13:16]] (both figurative); <span class="bdbLangHebrew">‎כּוֺכְבֵי נִשְׁפּוֺ‎</span> [[Job.3.9|Job 3:9]] ***the stars of its twilight*** (i.e. of night of Job's birth). — so <span class="bdbLangHebrew">‎וְעַדהָֿעֶרֶב‎</span> <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎מֵהַנּ‎</span> <span class="bdbLangHebrew">‎לְמָחֳרָתָם‎</span> [[1Sam.30|1Sam 30]] [[BDB#Klo|Klo]] [[BDB#Kit|Kit]] [[BDB#HPS|HPS]] (read <span class="bdbLangHebrew">‎לְהַחֲרִמָם‎</span> for <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎לְמָח‎</span>, [[BDB#We|We]] [[BDB#Kit|Kit]] [[BDB#Bu|Bu]]); [[BDB#The|Th]], after [[BDB#Lu|Luth]], below so [[BDB#Buhl|Buhl]]<sup>Kex</sup> [[BDB#SS|SS]]; Löhr allows either. 

**2.** ***morning twilight*** [[Job.7.4|Job 7:4]] (opposed to <span class="bdbLangHebrew">‎עֶרֶב‎</span>). apparently also [[Ps.119.147|Ps 119:147]].
