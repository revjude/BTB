 <span class="bdbLangHebrew">‎חַח‎</span> **noun masculine** ==hook, ring, fetter==; — <span class="bdbLangHebrew">‎חַח‎</span> absolute [[Exod.35.22|Exod 35:22]] suffix <span class="bdbLangHebrew">‎חַחִי‎</span> [[Isa.37.29|Isa 37:29]]; [[2Kgs.19.28|2Kgs 19:28]] plural <span class="bdbLangHebrew">‎חַחִים‎</span> [[Ezek.19.4|Ezek 19:4]] + 2 t. + [[Ezek.29.4|Ezek 29:4]] [[BDB#Qr|Qr]] ([[BDB#Kt|Kt]] erroneous <span class="bdbLangHebrew">‎חחיים‎</span>, see below); — 

**1.** ***hook, ring***, in nose of captive <span class="bdbLangHebrew">‎וְשַׂמְתִּי חַחִי‎</span> <span class="bdbLangHebrew">‎בְּאַפֶּ֑ךָ‎</span> [[Isa.37.29|Isa 37:29]] = [[2Kgs.19.28|2Kgs 19:28]] (compare <span class="bdbLangHebrew">‎בַּחֹחִים‎</span> [[2Chr.33.11|2Chr 33:11]], see  <span class="bdbLangHebrew">‎חוֺחַ‎</span> above); so of princes of Judah, under figure of young lions [[Ezek.19.4|Ezek 19:4]]; [[Ezek.19.9|Ezek 19:9]] (in v[[Ezek.19.9|Ezek 19:9]] transp:, so that <span class="bdbLangHebrew">‎בחחים‎</span> following <span class="bdbLangHebrew">‎הביא‎</span> as v[[Ezek.19.4|Ezek 19:4]] so [[BDB#Co|Co]]); in jaws of crocodile (metaphor of Pharaoh), with <span class="bdbLangHebrew">‎נתן‎</span> [[Ezek.29.4|Ezek 29:4]], compare [[Ezek.38.4|Ezek 38:4]] (but [[BDB#Sept|ᵐ5]] omits and sense opposes, compare [[BDB#Co|Co]] and — dubious — [[BDB#Da|Da]]). 

**2.** ***hook, ring*** as ornament, perhaps ***nose-ring*** (others ***brooch***, see [[BDB#VB|VB]]), [[Exod.35.22|Exod 35:22]] ([[BDB#P|P]]; \|\| <span class="bdbLangHebrew">‎נֶזֶם‎</span>, <span class="bdbLangHebrew">‎טַבַּעַת‎</span>, <span class="bdbLangHebrew">‎כּוּמָז‎</span>).
