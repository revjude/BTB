 <span class="bdbLangHebrew">‎תֹּ֫הוּ‎</span> **noun masculine**<sup>[[1Sam.12.21|1Sam 12:21]]</sup> ([[BDB#Albr.|Albr]]<sup>ZAW xvi (1896), 112</sup>) ==formlessness, confusion, unreality, emptiness== (primary meaning difficult to seize; [[BDB#Vrss|Vrss]] usually <span class="bdbLangGreek">Κενόν</span>, <span class="bdbLangGreek">οὐδέν</span>, <span class="bdbLangGreek">μάταιον</span>, ***inane, vacuum, vanum***; compare [[BDB#Lag|Lag]]<sup>Or. ii. 60</sup>; BN 144); — 

**1.** ***formlessness***, of primaeval earth [[Gen.1.2|Gen 1:2]] ([[BDB#P|P]]), of land reduced to primaeval chaos [[Jer.4.23|Jer 4:23]] (both + <span class="bdbLangHebrew">‎וָבֹהוּ‎</span> ***and voidness***), [[Isa.34.11|Isa 34:11]] <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎קַותֿֿ‎</span> (\|\| <span class="bdbLangHebrew">‎אַבְנֵי בֹהוּ‎</span>), [[Isa.45.18|Isa 45:18]] <span class="bdbLangHebrew">‎בְרָאָהּ‎</span> <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎לֹא ת‎</span> (\|\| <span class="bdbLangHebrew">‎לָשֶׁבֶת יְצָרָתּ‎</span>); [[Isa.24.10|Isa 24:10]] <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎קִרְיַתאתּ‎</span> ***city of chaos*** (of ruined city); = ***nothingness, empty space***, [[Job.26.7|Job 26:7]] <span class="bdbLangHebrew">‎תֹּלֶה אֶרֶץ‎</span> <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎עַלתּֿ‎</span>; of empty, trackless ***waste*** [[Deut.32.10|Deut 32:10]] (\|\| <span class="bdbLangHebrew">‎מִדְבָּר‎</span>), [[Job.6.18|Job 6:18]]; [[Job.12.24|Job 12:24]] = [[Ps.107.40|Ps 107:40]]. 

**2.** figurative of what is ***empty, unreal***, as idols [[1Sam.12.21|1Sam 12:21]] (collective: <span class="bdbLangHebrew">‎אֲשֶׁר‎</span> <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎אַחֲרֵי הַתּ‎</span> <span class="bdbLangHebrew">‎לֹא יוֺעִילוּ‎</span>), v[[1Sam.12.21|1Sam 12:21]]; [[Isa.41.29|Isa 41:29]] <span class="bdbLangHebrew">‎נִסְכֵּיהֶם‎</span> <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎רוּחַ וָת‎</span>, [[Isa.44.9|Isa 44:9]] (of idol-makers), groundless arguments or considerations, [[Isa.29.21|Isa 29:21]] <span class="bdbLangHebrew">‎צַדִּיק‎</span> <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎וַיַּטּוּ בַתּ‎</span>, [[Isa.59.4|Isa 59:4]] moral unreality or falsehood <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎בָּטוֺחַ עַלאתּ‎</span> (\|\| <span class="bdbLangHebrew">‎וְדַבֶּרשָֿׁוְא‎</span>); = ***a thing of nought*** (compare [[Eccl.41.10|Eccl 41:10]] <span class="bdbLangHebrew">‎מתהו אל תהו‎</span>), [[Isa.40.17|Isa 40:17]] (\|\| <span class="bdbLangHebrew">‎אַיִן‎</span> <span class="bdbLangHebrew">‎אֶפֶס‎</span>), v[[Isa.40.23|Isa 40:23]] <span class="bdbLangHebrew">‎עָשָׂה‎</span> <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎שֹׁפְטֵי אֶרֶץ כַּתּ‎</span> (\|\| <span class="bdbLangHebrew">‎לְאַיִן‎</span>), ***worthlessness*** [[Isa.49.19|Isa 49:19]] <span class="bdbLangHebrew">‎וְהֶבֶל כֹּחִי כִלֵּיתִי‎</span> <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎לְת‎</span> (\|\| <span class="bdbLangHebrew">‎לְרִיק יָגַעְתִּי‎</span>); as adverb accusative [[Isa.45.19|Isa 45:19]] I said not, <span class="bdbLangHebrew">‎תֹּהוּ בַקְּשׁוּנִי‎</span> seek me ***emptily, to no purpose.*** compare [[Isa.29.13|Isa 29:13]] [[BDB#Sept|ᵐ5]] <span class="bdbLangHebrew">‎וְתֹהוּ‎</span> for <span class="bdbLangHebrew">‎וַתְּהִי‎</span>.
