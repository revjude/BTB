 <span class="bdbLangHebrew">‎נָכְרִי‎</span> **adjective** ==foreign, alien== (Late Hebrew <span class="bdbLangHebrew">‎נָכְרִי‎</span> = ***Gentile***); — <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎נ‎</span> absolute masculine [[Zeph.1.8|Zeph 1:8]] +; feminine <span class="bdbLangHebrew">‎נָכְרִיָּה‎</span> [[Exod.2.22|Exod 2:22]] +; masculine plural <span class="bdbLangHebrew">‎נָכְרִים‎</span> [[Isa.2.6|Isa 2:6]] + 2 t. + [[Prov.20.16|Prov 20:16]] [[BDB#Kt|Kt]] (&gt; [[BDB#Qr|Qr]] <span class="bdbLangHebrew">‎נָכְרִיָּה‎</span>), + [[Prov.27.13|Prov 27:13]] (so read for [[BDB#MT|ᵑ0]] <span class="bdbLangHebrew">‎נָכְרִיָּה‎</span>); feminine plural <span class="bdbLangHebrew">‎נָכְרִיּוֺת‎</span> [[Gen.31.15|Gen 31:15]] +; — 

**1. a.** ***foreign:*** <span class="bdbLangHebrew">‎עַם נָכְרִי‎</span> [[Exod.21.8|Exod 21:8]] ([[BDB#E|E]]) ***a foreign*** (non-Israel) people, <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎אִישׁ נ‎</span> [[Deut.17.15|Deut 17:15]] (\|\| <span class="bdbLangHebrew">‎לֹא‎</span> <span class="bdbLangHebrew">‎אָתִיךָ‎</span>); <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎מַלְבּוּשׁ נ‎</span> [[Zeph.1.8|Zeph 1:8]] <span class="bdbLangHebrew">‎אֶרֶץ נָכְרִיָּה‎</span> [[Exod.2.22|Exod 2:22]] ([[BDB#J|J]]) ***foreign land***, so [[Exod.18.3|Exod 18:3]] ([[BDB#E|E]]); especially <span class="bdbLangHebrew">‎נָשִׁים נָכְרִיּוֺת‎</span> ***foreign*** (non-Israel) ***women*** [[1Kgs.11.18|1Kgs 11:18]]; [[Ezra.10.2|Ezra 10:2]]; [[Ezra.10.10|Ezra 10:10]]; [[Ezra.10.11|Ezra 10:11]]; [[Ezra.10.14|Ezra 10:14]]; [[Ezra.10.17|Ezra 10:17]]; [[Ezra.10.18|Ezra 10:18]]; [[Ezra.10.44|Ezra 10:44]]; [[Neh.13.26|Neh 13:26]]; [[Neh.13.27|Neh 13:27]]. 
$\quad$ $\quad$ **b.** as substantive <span class="bdbLangHebrew">‎נָכְרִי‎</span> ***foreigner*** (non-Israel) [[Judg.19.12|Judg 19:12]] (+ <span class="bdbLangHebrew">‎אֲשֶׁר לֹא מִבְּנֵי יִשְׂרָאֵל‎</span>), [[1Kgs.8.41|1Kgs 8:41]] (+ <span class="bdbLangHebrew">‎אֲשֶׁר לֹא מֵעַמְּךָ‎</span>), = [[2Chr.6.32|2Chr 6:32]] (+ ***id.***), [[1Kgs.8.43|1Kgs 8:43]] = [[2Chr.6.33|2Chr 6:33]]; [[Deut.14.21|Deut 14:21]] (\|\| <span class="bdbLangHebrew">‎גֵּר‎</span>), [[Deut.15.3|Deut 15:3]]; [[Deut.23.21|Deut 23:21]] (opposed to <span class="bdbLangHebrew">‎אָחִיךָ‎</span>), [[Deut.29.21|Deut 29:21]] (+ <span class="bdbLangHebrew">‎אֲשֶׁר יָבאֹ מֵאֶרֶץ רְחוֺקָה‎</span>); as predicate, [[2Sam.15.19|2Sam 15:19]] (\|\| <span class="bdbLangHebrew">‎גֹּלֶה‎</span>), <span class="bdbLangHebrew">‎נָכְרִיָּה‎</span> [[Ruth.2.10|Ruth 2:10]] plural <span class="bdbLangHebrew">‎נָכְרִים‎</span> ***foreigners*** [[Lam.5.2|Lam 5:2]] (<span class="bdbLangHebrew">‎זָרִים‎</span>), [[Obad.1.11|Obad 1:11]] (\|\| ***id.***); compare <span class="bdbLangHebrew">‎יַלְדֵי נָכְרִים‎</span> [[Isa.2.6|Isa 2:6]] feminine plural ***alien women*** [[Gen.31.15|Gen 31:15]] ([[BDB#E|E]]; i.e. not of one's father's family). 

**2.** <span class="bdbLangHebrew">‎נָכְרִיָּה‎</span> ***foreign woman***, as technical term, in Proverbs, for ***harlot*** (perhaps because harlots were originally chiefly foreigners): [[Prov.2.16|Prov 2:16]] (\|\| <span class="bdbLangHebrew">‎אִשָּׁה זָרָה‎</span>), [[Prov.7.5|Prov 7:5]] (\|\| ***id.***), [[Prov.5.20|Prov 5:20]] (\|\| <span class="bdbLangHebrew">‎זָרָה‎</span>), [[Prov.6.24|Prov 6:24]] (\|\| <span class="bdbLangHebrew">‎אֵשֶׁת רָ֑ע‎</span>), [[Prov.23.27|Prov 23:27]] (\|\| <span class="bdbLangHebrew">‎זוֺנָה‎</span>). — On [[Prov.20.16|Prov 20:16]]; [[Prov.27.13|Prov 27:13]] see below 

**3.** figurative ***unknown, unfamiliar:*** <span class="bdbLangHebrew">‎נָכְרִי הָיִיתִי בְּעֵינֵיהֶם‎</span> [[Job.19.15|Job 19:15]] ***an alien am I become in their eyes***; <span class="bdbLangHebrew">‎לִבְנֵי אִמִּי‎</span> <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎נ‎</span> [[Ps.69.9|Ps 69:9]] (\|\| <span class="bdbLangHebrew">‎מוּזָר הָיִיתִי לְאֶחָ֑י‎</span>); <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎אִישׁ נ‎</span> [[Eccl.6.2|Eccl 6:2]] of <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎י‎</span>'s judgment, <span class="bdbLangHebrew">‎נָכְרִיָּה עֲבֹדָתוֺ‎</span> [[Isa.28.21|Isa 28:21]] ***strange is his task!*** (\|\| <span class="bdbLangHebrew">‎זָר מַעֲשֵׂהוּ‎</span>); <span class="bdbLangHebrew">‎גֶּפֶן נָכְרִיָּה‎</span> [[Jer.2.21|Jer 2:21]] ***an alien vine*** (opposed to <span class="bdbLangHebrew">‎זֶרַע אֱמֶת‎</span>), figurative of degenerate Israel; as substantive, <span class="bdbLangHebrew">‎נָכְרִים‎</span> [[Prov.20.16|Prov 20:16]] [[BDB#Kt|Kt]] (&gt; [[BDB#Qr|Qr]] <span class="bdbLangHebrew">‎נָכְרִיָּה‎</span>), ***alens***, persons unknown to him (\|\| <span class="bdbLangHebrew">‎זָר‎</span>), so read also (for [[BDB#MT|ᵑ0]] <span class="bdbLangHebrew">‎נָכְרִיָּה‎</span>) \|\| [[Prov.27.13|Prov 27:13]].
