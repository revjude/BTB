 II. <span class="bdbLangHebrew">‎פּוּק‎</span> **verb Hiph\`il** ==bring out, furnish, promote== (Late Hebrew ***id.***; apparently akin to Aramaic <span class="bdbLangAramaic">‎נְפַק‎</span> ***go forth***, Aph\`el ***bring forth, produce***); — 

**1.** ***produce, furnish: Imperfect*** (jussive) <span class="bdbLangHebrew">‎וְתָפֵק לָרָעֵב‎</span> [[Isa.58.10|Isa 58:10]] with <span class="bdbLangHebrew">‎נַפְשְׁךָ‎</span> (read perhaps <span class="bdbLangHebrew">‎לַחְמְךָ‎</span> see [[BDB#Che|Che]]<sup>Hpt</sup>); ***Participle*** <span class="bdbLangHebrew">‎מְפִיקִים‎</span> <span class="bdbLangHebrew">‎מִזַּן אֶלזַֿן‎</span> [[Ps.144.13|Ps 144:13]] (see  <span class="bdbLangHebrew">‎זַן‎</span>). 

**2.** ***bring out, elicit, obtain***, with accusative; <span class="bdbLangHebrew">‎יָפִיק‎</span> [[Prov.3.13|Prov 3:13]], + <span class="bdbLangHebrew">‎מִן‎</span> person, <span class="bdbLangHebrew">‎וַיָּפֶק‎</span> [[Prov.8.35|Prov 8:35]]; [[Prov.18.22|Prov 18:22]] (all \|\| <span class="bdbLangHebrew">‎מצא‎</span>), [[Prov.12.2|Prov 12:2]]. 

**3.** <span class="bdbLangHebrew">‎זְמָמוֺ אַלתָּֿפֵק‎</span> [[Ps.140.9|Ps 140:9]] ***his device do not promote.***
