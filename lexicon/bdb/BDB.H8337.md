 <span class="bdbLangHebrew">‎שֵׁשׁ‎</span>, <span class="bdbLangHebrew">‎שִׁשָּׁה‎</span><sub>216</sub> **noun masculine** and **feminine** ==six==; — 

**masculine** <span class="bdbLangHebrew">‎שֵׁשׁ‎</span> (c. **noun feminine**) [[Gen.31.41|Gen 31:41]] + 120 t.; **feminine** <span class="bdbLangHebrew">‎שִׁשָּׁה‎</span> (with **noun masculine**) [[Gen.30.20|Gen 30:20]] + 39 t.; construct <span class="bdbLangHebrew">‎שֵׁ֫שֶׁת‎</span> [[Exod.23.12|Exod 23:12]] + 30 t. (6 + 10 = 16 are additional); — ***six*** (Hexateuch chiefly [[BDB#P|P]]): 

**1.** no other number: 
$\quad$ $\quad$ **a.** before plural noun 75 t.: ***e.g.*** <span class="bdbLangHebrew">‎שֵׁשׁ שָׁנִים‎</span> [[Exod.21.2|Exod 21:2]] ([[BDB#E|E]]) + 11 t., <span class="bdbLangHebrew">‎שֵׁשׁ אַמּוֺת‎</span> [[1Sam.17.4|1Sam 17:4]] + 8 t.; <span class="bdbLangHebrew">‎שֵׁשׁ בָּאַמָּה‎</span> [[1Kgs.6.6|1Kgs 6:6]] <span class="bdbLangHebrew">‎שִׁשָּׁה בָּנִים‎</span> [[Gen.30.20|Gen 30:20]] ([[BDB#E|E]]) + 2 t.; <span class="bdbLangHebrew">‎שֵׁשֶׁת יָמִים‎</span> [[Exod.23.12|Exod 23:12]] ([[BDB#E|E]]) + 14 t.; after noun (late) [[Exod.28.2|Exod 28:2]]; [[Josh.15.59|Josh 15:59]]; [[Josh.15.62|Josh 15:62]] (all [[BDB#P|P]]), [[1Chr.4.27|1Chr 4:27]]; [[1Chr.26.17|1Chr 26:17]]; [[Neh.5.18|Neh 5:18]] rarely noun omitted [[Lev.24.6|Lev 24:6]] ([[BDB#P|P]]) + (distributive [[2Sam.21.20|2Sam 21:20]] = [[1Chr.20.6|1Chr 20:6]]). 
$\quad$ $\quad$ **b.** = ordinal <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎שְׁנַת שֵׁשׁ לְחִזְק‎</span> [[2Kgs.18.10|2Kgs 18:10]] ([[BDB#Ges|Ges]]<sup>«GKC:134o»§ 134o</sup>). 

**2.** 16, c **noun feminine**: <span class="bdbLangHebrew">‎שֵׁשׁ עֶשְׂרֵה‎</span>, before <span class="bdbLangHebrew">‎נָ֑פֶשׁ‎</span> [[Gen.46.18|Gen 46:18]] ([[BDB#P|P]]), <span class="bdbLangHebrew">‎שָׁנָה‎</span> [[2Kgs.13.10|2Kgs 13:10]] + 9 t., <span class="bdbLangHebrew">‎בָּנוֺת‎</span> [[2Chr.13.21|2Chr 13:21]] after <span class="bdbLangHebrew">‎עָרִים‎</span> [[Josh.15.41|Josh 15:41]]; [[Josh.19.22|Josh 19:22]] ([[BDB#P|P]]); with **noun masculine** <span class="bdbLangHebrew">‎שִׁשָּׁה עָשָׂר‎</span>, before <span class="bdbLangHebrew">‎אֲדָנִים‎</span> [[Exod.26.25|Exod 26:25]]; [[Exod.36.30|Exod 36:30]] ([[BDB#P|P]]), also <span class="bdbLangHebrew">‎אֶלֶף‎</span>, 1000, [[Num.31.40|Num 31:40]] + ([[BDB#P|P]]); after <span class="bdbLangHebrew">‎בָּנִים‎</span> [[1Chr.4.27|1Chr 4:27]], etc.; = ordinal 16***th*** [[1Chr.24.14|1Chr 24:14]]; [[1Chr.25.23|1Chr 25:23]] +, <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎ע ‎</span><span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎יוֺם שׁ‎</span> [[2Chr.29.27|2Chr 29:27]]. 

**3.** 600 = <span class="bdbLangHebrew">‎שֵׁשׁ מֵאוֺת‎</span>, [[Gen.7.6|Gen 7:6]] ([[BDB#P|P]]) + 63 t.; 6000 = <span class="bdbLangHebrew">‎שֵׁשֶׁת אֲלָפִים‎</span> [[Num.3.34|Num 3:34]] ([[BDB#P|P]]) + 8 t.; <span class="bdbLangHebrew">‎אֶלֶף‎</span> <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎שׁ‎</span> [[Judg.20.15|Judg 20:15]] + 2 t. 

**4.** c. tens, <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎שׁ‎</span> following [[Gen.46.26|Gen 46:26]] + 13 t., precedes [[Num.31.44|Num 31:44]] ([[BDB#P|P]]). 

**5.** <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎שׁ‎</span> as round number, <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎חָמֵשׁ אוֺשֿׁ‎</span> [[2Kgs.13.19|2Kgs 13:19]], compare [[Job.5.19|Job 5:19]]; [[Prov.6.16|Prov 6:16]].
