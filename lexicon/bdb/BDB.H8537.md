 <span class="bdbLangHebrew">‎תֹּם‎</span> **noun [masculine]** ==completeness, integrity== ; — <span class="bdbLangHebrew">‎תֹּם‎</span> absolute [[Prov.10.20|Prov 10:20]] +; construct [[Job.4.6|Job 4:6]]; [[Ps.78.72|Ps 78:72]], construct also <span class="bdbLangHebrew">‎תָּםֿ‎</span> [[Prov.13.6|Prov 13:6]] + 4 t.; suffix <span class="bdbLangHebrew">‎תֻּמִּי‎</span> [[Ps.26.1|Ps 26:1]] + 11 t. suffix; plural <span class="bdbLangHebrew">‎תֻּמִּים‎</span> [[Exod.28.30|Exod 28:30]] + 3 t.; suffix <span class="bdbLangHebrew">‎תֻּמֶּיךָ‎</span> [[Deut.33.8|Deut 33:8]] ; — 

**1.** ***completeness, fullness :*** <span class="bdbLangHebrew">‎כְּתֻמָּם‎</span> [[Isa.47.9|Isa 47:9]] ***according to their full measure***; <span class="bdbLangHebrew">‎בְּעֶצֶם תֻּמּוֺ‎</span> [[Job.21.23|Job 21:23]] ***in his very completeness.*** 

**2.** ***innocence, simplicity:*** <span class="bdbLangHebrew">‎הלכים‎</span> <span class="bdbLangHebrew">‎לתמם‎</span> [[2Sam.15.11|2Sam 15:11]] <span class="bdbLangHebrew">‎מֶשַׁךְ בַּקֶּשֶׁת לְתֻמּוֺ‎</span> [[1Kgs.22.34|1Kgs 22:34]] = [[2Chr.18.33|2Chr 18:33]], i.e. without definite aim. 

**3.** ***integrity:*** <span class="bdbLangHebrew">‎תֹּם לֵבָב‎</span> ***integrity of mind*** [[Gen.20.5|Gen 20:5]]; [[Gen.20.6|Gen 20:6]] ([[BDB#E|E]]), [[1Kgs.9.4|1Kgs 9:4]]; [[Ps.78.72|Ps 78:72]]; [[Ps.101.2|Ps 101:2]] <span class="bdbLangHebrew">‎תָּםדֶּֿרֶךְ‎</span> [[Prov.13.6|Prov 13:6]]; [[Job.4.6|Job 4:6]] <span class="bdbLangHebrew">‎(הת)הלך בַּתֹּם‎</span> [[Prov.10.9|Prov 10:9]], <span class="bdbLangHebrew">‎בְּתֻמּוֺ‎</span> [[Prov.19.1|Prov 19:1]]; [[Prov.20.7|Prov 20:7]]; [[Prov.28.6|Prov 28:6]], <span class="bdbLangHebrew">‎בְּתֻמִּי‎</span> [[Ps.26.1|Ps 26:1]]; [[Ps.26.11|Ps 26:11]] <span class="bdbLangHebrew">‎הֹלְכֵי תֹם‎</span> [[Prov.2.7|Prov 2:7]] <span class="bdbLangHebrew">‎תֹּם וָישֶׁר‎</span> [[Ps.25.21|Ps 25:21]] <span class="bdbLangHebrew">‎בְּתֻמִּי‎</span> [[Ps.41.13|Ps 41:13]] <span class="bdbLangHebrew">‎כְּתֻמִּי‎</span> [[Ps.7.9|Ps 7:9]] <span class="bdbLangHebrew">‎מָעוֺז‎</span> <span class="bdbLangHebrew">‎לַתֹּם‎</span> [[Prov.10.29|Prov 10:29]]. 

**4.** <span class="bdbLangHebrew">‎תֻּמִּים‎</span> in <span class="bdbLangHebrew">‎אוּרִים וְתֻמִּים‎</span> (meaning dubious, see, <span class="bdbLangHebrew">‎אורים‎</span>) [[Deut.33.8|Deut 33:8]]; [[Exod.28.30|Exod 28:30]] ([[BDB#P|P]]) [[Lev.8.8|Lev 8:8]] ([[BDB#P|P]]), [[Ezra.2.63|Ezra 2:63]] = [[Neh.7.65|Neh 7:65]] + [[1Sam.14.41|1Sam 14:41]] (for <span class="bdbLangHebrew">‎תָּמִים‎</span>) [[BDB#We|We]] [[BDB#Dr|Dr]] [[BDB#Bu|Bu]] [[BDB#HPS|HPS]]; also [[Hos.3.4|Hos 3:4]] (for <span class="bdbLangHebrew">‎תרפים‎</span> [[BDB#Che|Che]]<sup>JQ.
