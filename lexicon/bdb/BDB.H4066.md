 I. <span class="bdbLangHebrew">‎מָדוֺן‎</span> **noun masculine**<sup>[[Prov.22.10|Prov 22:10]]</sup> ==strife, contention== — [[Ps.80.7|Ps 80:7]] + 9 t.; plural <span class="bdbLangHebrew">‎מִדְיָנִים‎</span> [[Prov.18.18|Prov 18:18]], <span class="bdbLangHebrew">‎מִדְ֯וָנִים‎</span> [[Prov.18.19|Prov 18:19]] + 7 t.; contracted into <span class="bdbLangHebrew">‎מְדָנִים‎</span> [[Prov.6.19|Prov 6:19]]; [[Prov.10.12|Prov 10:12]], <span class="bdbLangHebrew">‎מִדְ֯נִים‎</span> [[Prov.6.14|Prov 6:14]] construct <span class="bdbLangHebrew">‎מִדְיְנֵי‎</span> [[Prov.19.13|Prov 19:13]]. 

**1.** ***strife, contention***, almost wholly Proverbs: — [[Prov.18.19|Prov 18:19]]; [[Prov.23.29|Prov 23:29]]; [[Prov.26.20|Prov 26:20]] also [[Prov.17.14|Prov 17:14]] (\|\| <span class="bdbLangHebrew">‎רִיב‎</span>), [[Prov.22.10|Prov 22:10]] (\|\| <span class="bdbLangHebrew">‎דִּין וְקָלוֺן‎</span>); <span class="bdbLangHebrew">‎יִשָּׂא‎</span> <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎מ‎</span> [[Hab.1.3|Hab 1:3]] ***contention ariseth*** (\|\| <span class="bdbLangHebrew">‎ריב‎</span>); <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎שׁלּח מ‎</span> ***let loose strife*** [[Prov.6.14|Prov 6:14]]; [[Prov.16.28|Prov 16:28]] <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎יְגָרֶה מ‎</span> ***he stirreth up strife*** [[Prov.15.18|Prov 15:18]]; [[Prov.28.25|Prov 28:25]]; [[Prov.29.22|Prov 29:22]] note especially <span class="bdbLangHebrew">‎אִישׁ מִדְוָ֯נִים‎</span> [[Prov.26.21|Prov 26:21]] = ***a contentious, quarrelsome man***; oftener <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎אֵשֶׁת מ‎</span> = ***a contentious woman*** [[Prov.21.9|Prov 21:9]]; [[Prov.25.24|Prov 25:24]]; [[Prov.27.15|Prov 27:15]], compare <span class="bdbLangHebrew">‎וָכָ֑עַם‎</span> <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎אשׁת מ‎</span> [[Prov.21.19|Prov 21:19]]. 

**2.** ***object of contention*** [[Ps.80.7|Ps 80:7]], compare <span class="bdbLangHebrew">‎אִישׁ מָדוֺן‎</span> [[Jer.15.10|Jer 15:10]] (\|\| <span class="bdbLangHebrew">‎אִישׁ רִיב‎</span>). — On <span class="bdbLangHebrew">‎מָדוֺן‎</span> [[2Sam.21.20|2Sam 21:20]] [[BDB#Qr|Qr]], see  I. <span class="bdbLangHebrew">‎מִדָּה‎</span> below <span class="bdbLangHebrew">‎מדד‎</span>.
