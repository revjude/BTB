**1. substantive** ***daytime*** (rare) [[Jer.15.9|Jer 15:9]] <span class="bdbLangHebrew">‎בעוד יומם‎</span> while it is yet ***daytime***, [[Jer.33.20|Jer 33:20]] b <span class="bdbLangHebrew">‎לְבִלְתִּי הֱיוֺת יומם ולילה‎</span> ***daytime*** and night, v[[Jer.33.25|Jer 33:25]] <span class="bdbLangHebrew">‎ולילה‎</span> <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎בְּרִיתִי י‎</span> (? read <span class="bdbLangHebrew">‎יוֺם‎</span> as v[[Jer.33.20|Jer 33:20]] a); [[Ezek.30.16|Ezek 30:16]] <span class="bdbLangHebrew">‎צָרֵי יוֺמָם‎</span> foes of ***daytime***, i.e. coming by day (compare [[Jer.15.8|Jer 15:8]] but text apparently defective, see [[BDB#Sept|ᵐ5]] [[BDB#Sm|Sm]]); once (late) <span class="bdbLangHebrew">‎בְּיוֺמָם‎</span> (compare [[BDB#Tg|ᵑ7]] <span class="bdbLangHebrew">‎בְּיֵמָם‎</span> [[1Sam.25.16|1Sam 25:16]]; [[Job.5.14|Job 5:14]]), [[Neh.9.19|Neh 9:19]] (varied from [[Exod.13.21|Exod 13:21]]; [[Deut.1.33|Deut 1:33]] <span class="bdbLangHebrew">‎יומם‎</span>). — [[Num.9.21|Num 9:21]] the sense required is (during) ***a day*** and a night; read probably <span class="bdbLangHebrew">‎יוֺם‎</span> 

**2. adverb** ***in the daytime, by day***, [[Num.10.34|Num 10:34]]; [[Job.24.16|Job 24:16]] (but see <span class="bdbLangHebrew">‎חִתֵּם‎</span>), [[Isa.4.6|Isa 4:6]]; [[Ezek.12.3|Ezek 12:3]]; [[Ezek.12.4|Ezek 12:4]]; [[Ezek.12.7|Ezek 12:7]] mostly with <span class="bdbLangHebrew">‎לַיְלָה‎</span>, and then often in poetry = ***continually:*** so <span class="bdbLangHebrew">‎יוֺמָם וָלַיְלָה‎</span> [[Exod.13.21|Exod 13:21]]; [[Josh.1.8|Josh 1:8]]; [[1Kgs.8.59|1Kgs 8:59]]; [[Ps.1.2|Ps 1:2]]; [[Ps.32.4|Ps 32:4]] +, <span class="bdbLangHebrew">‎לַיְלָה וְיוֺמָם‎</span> [[Deut.28.66|Deut 28:66]]; [[Isa.34.10|Isa 34:10]]; [[Jer.14.17|Jer 14:17]], in parallel clauses (especially in poetry) [[2Sam.21.10|2Sam 21:10]]; [[Isa.21.8|Isa 21:8]]; [[Jer.31.35|Jer 31:35]]; [[Ps.22.3|Ps 22:3]]; [[Ps.42.9|Ps 42:9]]; [[Ps.91.5|Ps 91:5]]; [[Ps.121.6|Ps 121:6]] +. [[Ps.13.3|Ps 13:3]] ***by day*** yields a lame sense: either add <span class="bdbLangHebrew">‎וָלַיְלָה‎</span> ([[BDB#Sept|ᵐ5]] Del [[BDB#Gr|Gr]] [[BDB#Che|Ch]]), or read <span class="bdbLangHebrew">‎(יוֺם יוֺם) יֹם יֹם‎</span> [[BDB#Lag|Lag]]<sup>Nov.Psalt.Gr.Ed.Spec.13</sup> [[BDB#Now|Now]].
