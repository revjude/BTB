 <span class="bdbLangHebrew">‎שָׁפָל‎</span> **adjective** ==low==; — absolute <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎שׁ‎</span> [[2Sam.6.22|2Sam 6:22]] +, construct <span class="bdbLangHebrew">‎שְׁפַל‎</span> [[Ps.29.23|Ps 29:23]] +; feminine singular <span class="bdbLangHebrew">‎שְׁפָלָה‎</span> [[Ezek.17.14|Ezek 17:14]] +, construct <span class="bdbLangHebrew">‎שִׁפְלַת‎</span> v[[Ezek.17.6|Ezek 17:6]] plural <span class="bdbLangHebrew">‎שְׁפָלִים‎</span> [[Job.5.11|Job 5:11]] — 

**1.** ***low***, in height, of vine [[Ezek.17.6|Ezek 17:6]], tree v[[Ezek.17.24|Ezek 17:24]] (opposed to <span class="bdbLangHebrew">‎גָּבֹהַּ‎</span>); <span class="bdbLangHebrew">‎מִןהָֿעוֺר‎</span> <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎שׁ‎</span> [[Lev.13.20|Lev 13:20]] ***lower*** (deeper) ***than the skin*** (around it), so v[[Lev.13.21|Lev 13:21]] v[[Lev.13.26|Lev 13:26]], compare [[Lev.14.37|Lev 14:37]]. 

**2.** of a modest, unambitious kingdom [[Ezek.17.14|Ezek 17:14]]; [[Ezek.29.14|Ezek 29:14]]; [[Ezek.29.15|Ezek 29:15]] ***low*** in station [[Job.5.11|Job 5:11]] (opposed to <span class="bdbLangHebrew">‎שָֽׂגְבוּ‎</span>), [[Ezek.21.31|Ezek 21:31]] ([[BDB#Ges|Ges]]<sup>«GKC:90f»§ 90f</sup>.; read <span class="bdbLangHebrew">‎הַשְּׁפָלָה֣‎</span>). 

**3.** ***humiliated***, <span class="bdbLangHebrew">‎בְּעֵינָ֑י‎</span> <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎שׁ‎</span> [[2Sam.6.22|2Sam 6:22]] plural [[Mal.2.9|Mal 2:9]] (\|\| <span class="bdbLangHebrew">‎נִבְזִים‎</span>). 

**4.** ***lowly***, as substantive [[Ps.138.6|Ps 138:6]] (opposed to <span class="bdbLangHebrew">‎גָּבֹהַּ‎</span>), <span class="bdbLangHebrew">‎רוּחַ שְׁפָלִים‎</span> [[Isa.57.15|Isa 57:15]] (\|\| <span class="bdbLangHebrew">‎נִדְכָּ אִים‎</span>); <span class="bdbLangHebrew">‎שְׁפַל רוּחַ‎</span> v[[Isa.57.15|Isa 57:15]], [[Prov.29.23|Prov 29:23]] (opposed to <span class="bdbLangHebrew">‎גַּאֲוַת אָדָם‎</span>), + [[Prov.16.19|Prov 16:19]] [[BDB#Thes|Thes]] and others (but see check**Qal 2**).
