 <span class="bdbLangHebrew">‎צֶ֫מֶד‎</span> **noun masculine**<sup>[[1Kgs.19.19|1Kgs 19:19]]</sup> ==couple, pair==; — <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎צ‎</span> construct [[Judg.19.3|Judg 19:3]] +; suffix <span class="bdbLangHebrew">‎צִמְדּוֺ‎</span> [[Jer.51.23|Jer 51:23]] plural <span class="bdbLangHebrew">‎צְמָדִים‎</span> [[1Kgs.19.19|1Kgs 19:19]]; [[2Kgs.9.25|2Kgs 9:25]] (but see below); construct <span class="bdbLangHebrew">‎צִמְדֵּי‎</span> [[Isa.5.10|Isa 5:10]] ([[BDB#Ges|Ges]]<sup>«GKC:93m»§ 93m</sup>); — 

**1.** ***couple, pair***, usually of animals, <span class="bdbLangHebrew">‎חֲמֹרִים‎</span> <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎צ‎</span> [[Judg.19.3|Judg 19:3]]; [[Judg.19.10|Judg 19:10]]; [[2Sam.16.1|2Sam 16:1]] <span class="bdbLangHebrew">‎בָּקָר‎</span> (<span class="bdbLangHebrew">‎הַ‎</span>) <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎צ‎</span> ***span of oxen*** [[1Sam.11.7|1Sam 11:7]]; [[1Kgs.19.21|1Kgs 19:21]], plural of more than one span v[[1Kgs.19.19|1Kgs 19:19]] (ploughing), [[Job.1.3|Job 1:3]]; [[Job.42.12|Job 42:12]] <span class="bdbLangHebrew">‎צֶמֶד פְּרָדִים‎</span> [[2Kgs.5.17|2Kgs 5:17]] <span class="bdbLangHebrew">‎פָּרָשִׁים‎</span> <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎צ‎</span> [[Isa.21.7|Isa 21:7]]; [[Isa.21.9|Isa 21:9]] ***a pair of horsemen***; plural also of one pair of riders [[2Kgs.9.25|2Kgs 9:25]] (but read probably <span class="bdbLangHebrew">‎צֶמֶד‎</span>, so [[BDB#Kit|Kit]]); <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎צ‎</span> alone = ***span*** (of oxen) [[Jer.51.23|Jer 51:23]]. 

**2.** a measure of land (only square measure in OT) like ***acre***; originally apparently what a ***span*** can plough (in a day; see [[BDB#Now|Now]]<sup>Archaeology i. 202</sup> [[BDB#Benz|Benz]]<sup>Archaeology 208</sup>; compare modern ***faddan***, = what, can be ploughed in a season Schumacher<sup>ZPV xii (1889), 163 f.</sup>, Across Jordan 22, or in a day Bergheim<sup>PEQ, 1894, 192</sup>, Ency. Bib.<sup>ACRE</sup>), <span class="bdbLangHebrew">‎שָׂדֶה‎</span> <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎צ‎</span> [[1Sam.14.14|1Sam 14:14]] (on text see [[BDB#Dr|Dr]] [[BDB#HPS|HPS]]); <span class="bdbLangHebrew">‎צִמְדֵּיכֶֿרֶם‎</span> [[Isa.5.10|Isa 5:10]].
