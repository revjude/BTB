 <span class="bdbLangHebrew">‎שָׁרוֺן‎</span> **proper name, of a location** ==Sharon== (plausibly connected with √ <span class="bdbLangHebrew">‎ישׁר‎</span> by [[BDB#Thes|Thes]] [[BDB#Ges|Ges]]<sup>12</sup> and others, compare [[BDB#GASm|GASm]]<sup>Geogr. 52</sup>; = \*<span class="bdbLangHebrew">‎יְשָׁרוֺן‎</span>; Phoenician <span class="bdbLangHebrew">‎שרן‎</span>) — strictly **noun masculine**<sup>[[Isa.65.10|Isa 65:10]]</sup> plain, level; — 

**1.** <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎הַשּׁ‎</span> (with article), name of maritime plain on Mediterranean, from Joppa northward, noted for fertility, [[Isa.33.9|Isa 33:9]] (\|\| <span class="bdbLangHebrew">‎בָּשָׁן‎</span>, <span class="bdbLangHebrew">‎כַּרְמֶל‎</span>; [[BDB#Sept|ᵐ5]] <span class="bdbLangGreek">ὁ Σαρων</span>), compare <span class="bdbLangHebrew">‎הֲדַר כַּרְמֶל וְהַשָּׁרוֺן‎</span> [[Isa.35.2|Isa 35:2]]; [[Isa.65.10|Isa 65:10]] (\|\| <span class="bdbLangHebrew">‎עֵמֶק‎</span> <span class="bdbLangHebrew">‎עָכוֺר‎</span>; [[BDB#Sept|ᵐ5]] <span class="bdbLangGreek">τῷ δρυμῷ</span>); <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎חֲבַצֶּלֶת הַשּׁ‎</span> [[Song.2.1|Song 2:1]] ([[BDB#Sept|ᵐ5]] <span class="bdbLangGreek">ἄνθος τοῦ πεδίου</span>); <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎הַבָּקָר הָרֹעִים בַּשּׁ‎</span> [[1Chr.27.29|1Chr 27:29]] so also [[Josh.12.18|Josh 12:18]] where read <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎מֶלֶךְ אֲפֵק לַשּׁ‎</span>, so [[BDB#Di|Di]] [[BDB#We|We]]<sup>Sam 55</sup> [[BDB#Ges|Ges]]<sup>12 after</sup> [[BDB#Sept|ᵐ5]], compare [[BDB#RS|RS]]<sup>OTJC 2, 273, 435. — On this plain</sup> see [[BDB#GASm|GASm]]<sup>Geogr. 147 ff.</sup> [[BDB#Buhl|Buhl]]<sup>Geogr. § 65.</sup>
