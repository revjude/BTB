 <span class="bdbLangHebrew">‎מְנוּחָה‎</span>, <span class="bdbLangHebrew">‎מְנֻחָה‎</span> **noun feminine** ==resting-place, rest==; — absolute <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎מ‎</span> [[Mic.2.10|Mic 2:10]] +, <span class="bdbLangHebrew">‎מְנֻחָה‎</span> [[Gen.49.15|Gen 49:15]] (but see below), [[2Sam.14.17|2Sam 14:17]] suffix <span class="bdbLangHebrew">‎מְנוּחָתִי‎</span> [[Ps.95.11|Ps 95:11]] + 2 t., <span class="bdbLangHebrew">‎מְנוּחָתֶ֑ךָ‎</span> [[Ps.132.8|Ps 132:8]] = [[2Chr.6.41|2Chr 6:41]] (where read as in Psalms, for anomalous <span class="bdbLangHebrew">‎לְנוּחֶ֑ךָ‎</span> MT), <span class="bdbLangHebrew">‎מְנֻחָתוֺ‎</span> [[Isa.11.10|Isa 11:10]]; [[Zech.9.1|Zech 9:1]] plural <span class="bdbLangHebrew">‎מְנוּחֹת‎</span> [[Isa.32.18|Isa 32:18]], <span class="bdbLangHebrew">‎מְנֻחוֺת‎</span> [[Ps.23.2|Ps 23:2]] — 

**1.** ***resting-place*** [[Mic.2.10|Mic 2:10]]; [[Num.10.33|Num 10:33]] ([[BDB#J|J]][[BDB#E|E]]), [[Deut.12.9|Deut 12:9]] (+ <span class="bdbLangHebrew">‎הַנַּחֲלָה‎</span>) so [[Ps.95.11|Ps 95:11]] (<span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎י‎</span>'s resting- plural), compare [[Ps.132.8|Ps 132:8]] = [[2Chr.6.41|2Chr 6:41]] (see above), [[Ps.132.14|Ps 132:14]]; [[Isa.66.1|Isa 66:1]] (\|\| <span class="bdbLangHebrew">‎בַּיִת אֲשֶׁר תִּבְנוּ לִי‎</span>), compare <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎לַאֲרוֺן וגו‎</span> <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎בֵּית מ‎</span> [[1Chr.28.2|1Chr 28:2]] also [[Isa.11.10|Isa 11:10]] probably also [[Gen.49.15|Gen 49:15]] (\|\| <span class="bdbLangHebrew">‎טוֺב הָאָרֶץ‎</span> here is apparently substantive, [[BDB#Sam|⅏]] reads adjective <span class="bdbLangHebrew">‎טוֺבָה‎</span>; but read perhaps <span class="bdbLangHebrew">‎מְנֻחהֹ‎</span>, ***his resting-place***, from <span class="bdbLangHebrew">‎מָנוֺחַ‎</span>, Ball<sup>Hpt</sup> [[BDB#Holz|Holz]] after [[BDB#Syr|ᵑ6]]); <span class="bdbLangHebrew">‎מְנוּחֹת שַׁאֲנַנּוֺת‎</span> [[Isa.32.18|Isa 32:18]] (+ <span class="bdbLangHebrew">‎נְוֵה שָׁלוֺם‎</span>, <span class="bdbLangHebrew">‎מִשְׁכְּנוֺת מִבְטַחִים‎</span>); of <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎י‎</span>'s word [[Zech.9.1|Zech 9:1]] resting-place on journey, in <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎שַׂר מ‎</span> [[Jer.51.59|Jer 51:59]] ([[BDB#Ew|Ew]] [[BDB#Gf|Gf]] [[BDB#Gie|Gie]] and others); figurative for security, assurance [[2Sam.14.17|2Sam 14:17]] (taken from ***resting***, equilibrium, of scale according to [[BDB#Klo|Klo]]). 

**2.** ***rest, quietness***, plural <span class="bdbLangHebrew">‎מֵי מְנֻחוֺת‎</span> [[Ps.23.2|Ps 23:2]] = refreshment [[Isa.28.12|Isa 28:12]] (\|\| <span class="bdbLangHebrew">‎מַרְגֵּעָה‎</span>); ***rest*** (from enemies) = peace [[1Kgs.8.56|1Kgs 8:56]] (compare [[1Kgs.5.18|1Kgs 5:18]]), <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎אִישׁ מ‎</span> [[1Chr.22.9|1Chr 22:9]] ***a man of peace*** (compare <span class="bdbLangHebrew">‎הֵנִיחַ‎</span> v<sup>b</sup>); = relief from sorrow [[Jer.45.3|Jer 45:3]] ***condition of rest*** and security attained by marriage [[Ruth.1.9|Ruth 1:9]] (= <span class="bdbLangHebrew">‎מָנוֺחַ‎</span> [[Ruth.3.1|Ruth 3:1]]). — <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎מ‎</span> [[Judg.20.43|Judg 20:43]] is perhaps proper name, compare [[BDB#GFM|GFM]].
