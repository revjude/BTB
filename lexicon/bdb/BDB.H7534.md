 <span class="bdbLangHebrew">‎רַק‎</span> 

**1. adjective** ==thin==; — feminine plural <span class="bdbLangHebrew">‎רַקּוֺת‎</span>, of kine, [[Gen.41.19|Gen 41:19]]; [[Gen.41.20|Gen 41:20]]; [[Gen.41.27|Gen 41:27]] (\|\| v[[Gen.41.3|Gen 41:3]] v[[Gen.41.4|Gen 41:4]] <span class="bdbLangHebrew">‎דַּקּוֺת‎</span>). 

**2.**<sub>109</sub> **adverb** with restrictive force, ==only, altogether, surely== (synonym <span class="bdbLangHebrew">‎אַךְ‎</span>); — 
$\quad$ $\quad$ **a.** ***only***, [[Gen.14.24|Gen 14:24]]; [[Gen.41.40|Gen 41:40]] <span class="bdbLangHebrew">‎רַק הַכִּסֵּא אֶגְדַּל מִמֶּךָּ‎</span> ***only*** as regards the throne, etc., [[Gen.47.22|Gen 47:22]] <span class="bdbLangHebrew">‎רַק אַדְמַת הַכֹּהֲנִים‎</span> <span class="bdbLangHebrew">‎לֹא קָנָה‎</span>, [[Gen.50.8|Gen 50:8]]; [[Exod.8.5|Exod 8:5]] <span class="bdbLangHebrew">‎רַק בַּיְאֹר תִּשָּׁאַרְנָה‎</span>, [[Exod.10.17|Exod 10:17]] <span class="bdbLangHebrew">‎וְיָסֵר‎</span> <span class="bdbLangHebrew">‎מֵעָלַי רַק הַמָּוֶת הַזֶּה‎</span> ***only*** this death, [[Deut.2.35|Deut 2:35]] (compare [[Deut.20.14|Deut 20:14]]; [[Josh.8.2|Josh 8:2]]; [[Josh.8.27|Josh 8:27]]), [[Josh.3.11|Josh 3:11]]; [[Judg.6.39|Judg 6:39]] (compare <span class="bdbLangHebrew">‎אַךְ‎</span> ***ib.*** [[Gen.18.32|Gen 18:32]]; [[Exod.10.17|Exod 10:17]]), [[Exod.11.34|Exod 11:34]] (circumstantial clause) <span class="bdbLangHebrew">‎רַק הִיא יְחִידָה‎</span> (there being) ***only*** she, an only one, [[1Sam.1.13|1Sam 1:13]]; [[Amos.3.2|Amos 3:2]] <span class="bdbLangHebrew">‎רק אתכם ידעתי‎</span> ***only*** you have I known, etc., [[Job.1.15|Job 1:15]]; [[Job.1.16|Job 1:16]]; [[Job.1.17|Job 1:17]]; [[Job.1.19|Job 1:19]] <span class="bdbLangHebrew">‎וָאִמָּֽלְטָה‎</span> <span class="bdbLangHebrew">‎רַק אֲנִי לְבַדִּי‎</span> ***I only***, [[Ps.91.8|Ps 91:8]] + often Once strengthening <span class="bdbLangHebrew">‎אַךְ‎</span>, [[Num.12.2|Num 12:2]] <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎הֲרַק אַךְבְּֿמשֶׁה דִּבֶּר י‎</span>. And separated (as sometimes in English) from the word actually emphasized, [[Prov.13.10|Prov 13:10]] <span class="bdbLangHebrew">‎רִק בְּזָדוֺן יִתֵּן מַצָּה‎</span> by pride there ***only*** cometh [<span class="bdbLangHebrew">‎נָתַן‎</span> **1 z**] ==contention.== 
$\quad$ $\quad$ **b.** prefixed to ***sentences***, to add a limitation on something previously expressed (or implied), [[Gen.19.8|Gen 19:8]] ***only*** to these men do nothing, [[Gen.24.8|Gen 24:8]] <span class="bdbLangHebrew">‎רִק אֶתבְּֿנִי‎</span> <span class="bdbLangHebrew">‎לֹאתָֿהֵ֫נָּה‎</span>, [[Exod.8.24|Exod 8:24]] I will let you go . . ., ***only*** go not far, v.[[Exod.8.25|Exod 8:25]]; [[Num.20.19|Num 20:19]] ***only*** — it is nothing — let me pass through on my feet (compare [[Deut.2.28|Deut 2:28]]); especially in Deuteronomic writers, as [[Deut.10.15|Deut 10:15]]; [[Deut.12.15|Deut 12:15]]; [[Deut.20.16|Deut 20:16]]; [[1Kgs.3.2|1Kgs 3:2]]; [[1Kgs.3.3|1Kgs 3:3]] (compare [[2Kgs.12.4|2Kgs 12:4]]; [[2Kgs.14.4|2Kgs 14:4]]; [[2Kgs.15.4|2Kgs 15:4]]; [[2Kgs.15.35|2Kgs 15:35]]), [[2Kgs.8.19|2Kgs 8:19]]; [[2Kgs.11.13|2Kgs 11:13]]; [[2Kgs.15.14|2Kgs 15:14]]; [[2Kgs.15.23|2Kgs 15:23]]; [[2Kgs.3.2|2Kgs 3:2]] (compare [[2Kgs.14.3|2Kgs 14:3]]; [[2Kgs.17.2|2Kgs 17:2]]), etc., [[Isa.4.1|Isa 4:1]] emphasizing a command, [[Deut.4.9|Deut 4:9]]; [[Deut.12.16|Deut 12:16]]; [[Deut.12.23|Deut 12:23]]; [[Josh.1.7|Josh 1:7]]; [[Josh.1.18|Josh 1:18]]; [[Josh.6.18|Josh 6:18]]; [[Josh.13.6|Josh 13:6]]; [[Josh.22.5|Josh 22:5]] (all [[BDB#D|D]]<sup>2</sup>). 
$\quad$ $\quad$ **c.** emphasizing single words, especially adjectives, ***only*** = ***nought but, altogether***, [[Gen.6.5|Gen 6:5]] <span class="bdbLangHebrew">‎יֵצֶר מַחְשְׁבוֺת לִבּוֺ רִק רִע‎</span> is ***only*** evil, i.e. ***exclusively*** evil, ***nought but*** evil, [[Gen.26.29|Gen 26:29]] <span class="bdbLangHebrew">‎עָשִׂינוּ עִמְּךָ רִק טוֺב‎</span> ***nothing but*** good, [[Deut.28.33|Deut 28:33]] <span class="bdbLangHebrew">‎וְהָיִיתָ רִק עָשׁוּק‎</span>, [[Isa.28.19|Isa 28:19]] <span class="bdbLangHebrew">‎רִק זְוָעָה‎</span> it shall be ***nought but*** terror to, etc., [[1Kgs.14.8|1Kgs 14:8]] (compare <span class="bdbLangHebrew">‎אַךְ‎</span> **2 b** <span class="bdbLangGreek">β</span>); followed by an adverb [[Deut.28.13|Deut 28:13]] <span class="bdbLangHebrew">‎וְהָיִיתָ רִק לְמַ֫עְלָה‎</span> above ***only***; followed by a verb, [[Judg.14.16|Judg 14:16]] thou dost ***but*** hate me. 
$\quad$ $\quad$ **d.** after a negative, ***save, except*** (synonym <span class="bdbLangHebrew">‎כִּי אִם‎</span> **2 a**), [[1Kgs.8.9|1Kgs 8:9]] ( = [[2Chr.5.10|2Chr 5:10]]) <span class="bdbLangHebrew">‎אֵין בָּאָרוֺן רִק שְׁנֵי לֻהוֺת הָאֲבָנִים‎</span>, [[2Chr.15.5|2Chr 15:5]]; [[2Chr.22.16|2Chr 22:16]] ( = [[2Chr.18.15|2Chr 18:15]]), [[2Kgs.17.18|2Kgs 17:18]]. 
$\quad$ $\quad$ **e.** with an affirmative, asseverative force, ***only, altogether = surely***, [[Gen.20.11|Gen 20:11]] <span class="bdbLangHebrew">‎רַק אֵין יִרְאַת אלהים במקום הזה‎</span>, [[Deut.4.6|Deut 4:6]] (so [[BDB#EV|EV]]; but [[BDB#Ges|Ges]] populus ***mere*** sapiens, so [[BDB#Di|Di]]), [[1Kgs.21.25|1Kgs 21:25]]; [[2Chr.28.10|2Chr 28:10]]; [[Ps.32.6|Ps 32:6]] <span class="bdbLangHebrew">‎רִק לְשֶׁטֶף מים רבים אליו‎</span> <span class="bdbLangHebrew">‎לא יַגִּ֑יעוּ‎</span>. **feminine** <span class="bdbLangHebrew">‎רִק אִם‎</span> (<span class="bdbLangHebrew">‎רִק‎</span> prefixed for emphasis) ***if only, provided only***, [[Deut.15.5|Deut 15:5]] <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎רִק אִם שׁמוע תשׁמע וג‎</span>, [[1Kgs.8.25|1Kgs 8:25]] ( = [[2Chr.6.16|2Chr 6:16]]) <span class="bdbLangHebrew">‎רִק אִם ישׁמרו בניך דרכם‎</span> <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎ללכת וג‎</span>, [[2Kgs.21.8|2Kgs 21:8]] ( = [[2Chr.33.8|2Chr 33:8]]).
