 II. [<span class="bdbLangHebrew">‎נָוָה‎</span>] **noun feminine** ==pasture, meadow==; — plural construct <span class="bdbLangHebrew">‎נְוֺת‎</span> [[Zeph.2.6|Zeph 2:6]], usually <span class="bdbLangHebrew">‎נְאוֺת‎</span> [[Ps.23.2|Ps 23:2]] + 11 t.; — 

**1.** ***pasture, meadow:*** <span class="bdbLangHebrew">‎הָרֹעִים‎</span> <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎נ‎</span> [[Amos.1.2|Amos 1:2]], compare [[Zeph.2.6|Zeph 2:6]] (\|\| <span class="bdbLangHebrew">‎גִּדְרוֺת צאֹן‎</span>; see also [<span class="bdbLangHebrew">‎כָּרה‎</span>] p. 500 above); especially <span class="bdbLangHebrew">‎נְאוֺת (הַ)מִּדְבָּר‎</span> [[Jer.9.9|Jer 9:9]]; [[Jer.23.10|Jer 23:10]]; [[Ps.65.13|Ps 65:13]]; [[Joel.1.19|Joel 1:19]]; [[Joel.1.20|Joel 1:20]]; [[Joel.2.22|Joel 2:22]] <span class="bdbLangHebrew">‎דֶּשֶׁא‎</span> <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎נ‎</span> [[Ps.23.2|Ps 23:2]] ***grassy pastures***; <span class="bdbLangHebrew">‎אֱלֹהִים‎</span> <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎נ‎</span> [[Ps.83.13|Ps 83:13]] ***pastures of God***, i.e. the land of Canaan, <span class="bdbLangHebrew">‎יַעֲקֹב‎</span> <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎נ‎</span> [[Lam.2.2|Lam 2:2]] <span class="bdbLangHebrew">‎הַשָּׁלוֺם‎</span> <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎נ‎</span> [[Jer.25.37|Jer 25:37]] ***meadows of pace***; <span class="bdbLangHebrew">‎חָמָס‎</span> <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎נ‎</span> [[Ps.74.20|Ps 74:20]] apparently ***habitations*** (?) ***of violence***, but read probably <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎גַּאֲוָה וְח‎</span>, so [[BDB#Bi|Bi]] [[BDB#Che|Che]] (see [[Ps.73.6|Ps 73:6]]). — [[Jer.6.2|Jer 6:2]] see following; [[Job.8.6|Job 8:6]] see  I. <span class="bdbLangHebrew">‎נָוֶה‎</span>.
