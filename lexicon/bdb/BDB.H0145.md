 <span class="bdbLangHebrew">‎אֶ֫דֶר‎</span> **noun [masculine]**

**2.** mangle, cloak (as ***wide***) [[Mic.2.8|Mic 2:8]] (\|\| <span class="bdbLangHebrew">‎שַׂלְמָה‎</span>), but read <span class="bdbLangHebrew">‎אדרת‎</span> (<span class="bdbLangHebrew">‎ת‎</span> lost before following <span class="bdbLangHebrew">‎ת‎</span>), so WRS<sup>Proph 427</sup>.
