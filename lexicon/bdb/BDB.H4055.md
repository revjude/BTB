 [<span class="bdbLangHebrew">‎מַד‎</span>] **noun masculine**<sup>[[1Sam.4.12|1Sam 4:12]]</sup> ==measure==, also ==cloth garment== (as ***extended, wide***); — suffix <span class="bdbLangHebrew">‎מַדּוֺ‎</span> [[Ps.109.18|Ps 109:18]] <span class="bdbLangHebrew">‎מִדּוֺ‎</span> [[Lev.6.3|Lev 6:3]]; [[2Sam.20.8|2Sam 20:8]] (<span class="bdbLangHebrew">‎מִדָּהּ‎</span> [[Job.11.9|Job 11:9]], see below <span class="bdbLangHebrew">‎מִדָּה‎</span>); plural <span class="bdbLangHebrew">‎מִדִּין‎</span> [[Judg.5.10|Judg 5:10]] (see below), [[2Sam.21.20|2Sam 21:20]] [[BDB#Kt|Kt]] ([[BDB#Qr|Qr]] <span class="bdbLangHebrew">‎מָדוֺן‎</span>; read perhaps <span class="bdbLangHebrew">‎מִדָּה‎</span>, see [[1Chr.20.6|1Chr 20:6]], compare [[BDB#Dr|Dr]]); suffix <span class="bdbLangHebrew">‎מִדַּיִךְ‎</span> [[Jer.13.25|Jer 13:25]] <span class="bdbLangHebrew">‎מַדָּיו‎</span> [[1Sam.4.12|1Sam 4:12]] + 4 t.; — 

**1.** ***measure*** (only plural) = apportionment [[Jer.13.25|Jer 13:25]], <span class="bdbLangHebrew">‎מְנָָֽתמִֿדַּיִךְ‎</span> (of lot of Judah, \|\| <span class="bdbLangHebrew">‎גּוֺרָל‎</span>); compare literally [[2Sam.21.20|2Sam 21:20]] [[BDB#Kt|Kt]] (but see  <span class="bdbLangHebrew">‎מִדָּה‎</span>). 

**2.** perhaps ***cloth, carpet*** (for sitting on), [[Judg.5.10|Judg 5:10]] (so most, conjecture from context; on plural in <span class="bdbLangHebrew">‎ן‎</span>, see [[BDB#Ges|Ges]]<sup>«GKC:87»§ 87</sup>, l. a). 

**3.** ***garment***, outer garment [[Judg.3.16|Judg 3:16]]; [[1Sam.4.12|1Sam 4:12]], compare (simile) [[Ps.109.18|Ps 109:18]] of fighting-attire [[1Sam.17.38|1Sam 17:38]] (with verb <span class="bdbLangHebrew">‎הִלְבִּשׁ‎</span>), v[[1Sam.17.39|1Sam 17:39]]; [[1Sam.18.4|1Sam 18:4]]; [[2Sam.20.8|2Sam 20:8]] (\|\| <span class="bdbLangHebrew">‎לְבוּשׁ‎</span>); of priest's garment <span class="bdbLangHebrew">‎וְלָבַשׁ הַכֹּהֵן מִדּוֺ בַד‎</span> [[Lev.6.3|Lev 6:3]] compare [<span class="bdbLangHebrew">‎מָדוּ‎</span>, <span class="bdbLangHebrew">‎מַדְוֶה‎</span>] below <span class="bdbLangHebrew">‎מדה‎</span>.
