 I. <span class="bdbLangHebrew">‎חָרַץ‎</span> **verb** ==cut, sharpen, decide== (Late Hebrew ***id., cut in, decide***, Aramaic in derivatives; Assyrian <abbr title="transliteration"><u>***—arâƒu***</u></abbr>, dig, decide, <abbr title="transliteration"><u>***—ariƒu***</u></abbr>, ***trench***, [[BDB#Zehnpf|Zehnpf]]<sup>BAS i. 502</sup>; Phoenician <span class="bdbLangHebrew">‎חרץ‎</span> ***decision***, [[BDB#Hoffm|Hoffm]]<sup>AGG xxxvi, May 1889, 11</sup>) — 

**Qal** ***Perfect*** 3 masculine singular <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎ח‎</span> [[Josh.10.21|Josh 10:21]] <span class="bdbLangHebrew">‎חָרָ֑צְתָּ‎</span> [[1Kgs.20.40|1Kgs 20:40]] ***Imperfect*** <span class="bdbLangHebrew">‎יֶחֱרַץֿ‎</span> [[Exod.11.7|Exod 11:7]] 2 masculine singular <span class="bdbLangHebrew">‎תֶּחֱרָ֑ץ‎</span> [[2Sam.5.24|2Sam 5:24]] ***Passive participle*** <span class="bdbLangHebrew">‎חָרוּץ‎</span> [[Isa.10.22|Isa 10:22]]; [[Lev.22.22|Lev 22:22]] plural <span class="bdbLangHebrew">‎חֲרוּצִים‎</span> [[Job.14.5|Job 14:5]] (see also <span class="bdbLangHebrew">‎חָרוּץ‎</span> below); — 

**1.** ***cut, mutilate*** [[Lev.22.22|Lev 22:22]] (<span class="bdbLangHebrew">‎עַוֶּרֶת אוֺ שָׁבוּר‎</span> <span class="bdbLangHebrew">‎אוֺחָֿרוּץ אוֺ יַבֶּלֶת‎</span>). 

**2.** ***sharpen***, figurative the tongue, <span class="bdbLangHebrew">‎כֶּֿלֶב לְשֹׁנוֺ לְ‎</span><span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎לֹא יֶח‎</span> [[Exod.11.7|Exod 11:7]] ([[BDB#J|J]]), i.e. utter no sound against Israel, [[Josh.10.21|Josh 10:21]]. 

**3.** ***decide*** [[1Kgs.20.40|1Kgs 20:40]] (absolute); so participle pass <span class="bdbLangHebrew">‎חרוצים ימיו‎</span> [[Job.14.5|Job 14:5]] ***his days are determined, fixed*** (\|\| <span class="bdbLangHebrew">‎מִסְפַּר חדשׁיו‎</span>), [[Isa.10.22|Isa 10:22]] <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎כִּלָּיוֺן ח‎</span>; ***act with decision*** [[2Sam.5.24|2Sam 5:24]]. **Niph\`al** ***Participle*** <span class="bdbLangHebrew">‎נֶחֱרָצָה‎</span> [[Isa.10.23|Isa 10:23]] + 3 t.; <span class="bdbLangHebrew">‎נֶחֱרֶצֶת‎</span> [[Dan.9.26|Dan 9:26]] (both these forms infinitive construct according to [[BDB#Ba|Ba]]<sup>NB 90</sup>); ***decisive*** <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎כָּלָה וְנֶח‎</span> [[Isa.10.23|Isa 10:23]]; [[Isa.28.22|Isa 28:22]]; [[Dan.9.27|Dan 9:27]] a consumption and ***strict decision*** (i.e. that which is strictly determined), <span class="bdbLangHebrew">‎נֶעֱשָׂ֑תָה‎</span> <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎כי נ‎</span> [[Dan.11.36|Dan 11:36]] <span class="bdbLangHebrew">‎שֹׁמֵמוֺת‎</span> <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎נח‎</span> [[Dan.9.26|Dan 9:26]] ***strict determining of desolation***.
