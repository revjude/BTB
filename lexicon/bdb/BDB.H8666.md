 [<span class="bdbLangHebrew">‎תְּשׁוּבָה‎</span>] **noun feminine** ==return, answer==; — 

**1.** suffix <span class="bdbLangHebrew">‎וּתְשֻׁבָתוֺ הָרָמָ֫תָה‎</span> [[1Sam.7.17|1Sam 7:17]] ***and his return was*** (=he returned) ***to Rama.*** 

**2.** especially construct <span class="bdbLangHebrew">‎לִתְשׁוּבַת‎</span> <span class="bdbLangHebrew">‎הַשָּׁנָה‎</span> ***at the return of the year***, i.e. of spring, [[2Sam.11.1|2Sam 11:1]]; [[1Kgs.20.22|1Kgs 20:22]]; [[1Kgs.20.26|1Kgs 20:26]]; [[2Chr.36.10|2Chr 36:10]], <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎הַשּׁ‎</span> <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎לְעֵת תְּשׁ‎</span> [[1Chr.20.1|1Chr 20:1]] (\|\| [[2Sam.11.1|2Sam 11:1]]). 

**3.** ***answer***, plural absolute <span class="bdbLangHebrew">‎תְּשֻׁבֹת‎</span> [[Job.34.36|Job 34:36]], suffix <span class="bdbLangHebrew">‎תְּשׁוּבֹתֵיכֶם‎</span> [[Job.21.34|Job 21:34]].
