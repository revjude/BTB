 [<span class="bdbLangHebrew">‎חָבָה‎</span>] **verb** ==withdraw, hide==, compare <span class="bdbLangHebrew">‎חבא‎</span>; — 

**Qal** ***Imperative*** masculine singular <span class="bdbLangHebrew">‎חֲבִי‎</span> [[Isa.26.20|Isa 26:20]] (Aramaic form, [[BDB#Ol|Ol]]<sup>§ 235, 3</sup> [[BDB#Ew|Ew]]<sup>§ 224.c</sup> [[BDB#Du|Du]]; [[BDB#Di|Di]] reads <span class="bdbLangHebrew">‎חֲבֵּי‎</span> = <span class="bdbLangHebrew">‎חֲבֵה‎</span>), ***withdraw*** into privacy (\|\| <span class="bdbLangHebrew">‎בּאֹ בַחֲדָרֶיךָ‎</span>). **Niph\`al** ***Perfect*** <span class="bdbLangHebrew">‎וְנֶחְבָּה‎</span> consecutive [[Jer.49.10|Jer 49:10]] <span class="bdbLangHebrew">‎לְנַחְבֵּתֶם‎</span> consecutive [[Josh.2.16|Josh 2:16]] ([[BDB#J|J]][[BDB#E|E]]); ***Infinitive*** <span class="bdbLangHebrew">‎לְהֵחָבֵה‎</span> (as <span class="bdbLangHebrew">‎א‎</span><span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎ל‎</span>) [[1Kgs.22.25|1Kgs 22:25]]; [[2Kgs.7.12|2Kgs 7:12]] ***withdraw, hide oneself***, literal [[Josh.216|Josh 216]] ([[BDB#J|J]][[BDB#E|E]]), followed by <span class="bdbLangHebrew">‎שָׁמָּה‎</span>; [[2Kgs.7.12|2Kgs 7:12]] with <span class="bdbLangHebrew">‎בְּ‎</span> local; [[1Kgs.22.25|1Kgs 22:25]] of Esau = Edom, ***hiding*** from judgment of <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎י‎</span>, <span class="bdbLangHebrew">‎וְנֶחְבָּה לֹא יוּכָּ֑ל‎</span> [[Jer.49.10|Jer 49:10]], as pointed, meant no doubt as perfect, but construction then hard; read rather infinitive absolute <span class="bdbLangHebrew">‎נַחְבֹּה‎</span>; so [[BDB#Gf|Gf]] [[BDB#Gie|Gie]] and others (see [[BDB#Kö|Kö]]<sup>i. 624</sup>).
