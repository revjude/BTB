 <span class="bdbLangHebrew">‎מַשְׂאֵת‎</span> **noun feminine** ==uprising, utterance, burden, portion== (on formation compare [[BDB#Ol|Ol]]<sup>§ 109 a</sup> [[BDB#Sta|Sta]]<sup>§ 272 b</sup> [[BDB#Kö|Kö]]<sup>ii. 1, 183</sup>; Phoenician (Pun) has <span class="bdbLangHebrew">‎משאת‎</span> = ***payment, tax*** see [[BDB#Ldzb|Lzb]]<sup>326</sup>); — absolute <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎מ‎</span> [[Judg.20.40|Judg 20:40]] +; construct <span class="bdbLangHebrew">‎מַשְׂאַת‎</span> [[Gen.43.34|Gen 43:34]] +; plural absolute <span class="bdbLangHebrew">‎מַשְׂאֹת‎</span> [[Gen.43.34|Gen 43:34]], + (probably) <span class="bdbLangHebrew">‎מַשְׂאוֺת‎</span> [[Lam.2.14|Lam 2:14]] (see below); construct <span class="bdbLangHebrew">‎מַשְׂאֹת‎</span> [[Gen.43.34|Gen 43:34]] — 

**1.** ***that which rises, uprising, uplifting:*** 
$\quad$ $\quad$ **a.** <span class="bdbLangHebrew">‎הֶעָשָׁן‎</span> <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎מ‎</span> [[Judg.20.38|Judg 20:38]] ***uprising of smoke***, compare (<span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎הֶע‎</span> omitted) v[[Judg.20.40|Judg 20:40]]. 
$\quad$ $\quad$ **b.** in general = signal [[Jer.6.1|Jer 6:1]] (= <span class="bdbLangHebrew">‎נֵס‎</span> [[BDB#Gf|Gf]].). 
$\quad$ $\quad$ **c.** ***uplifting*** of hands (<span class="bdbLangHebrew">‎כַּפַּי‎</span>) in prayer [[Ps.141.2|Ps 141:2]] (\|\| <span class="bdbLangHebrew">‎תְּפִלָּה‎</span>). 

**2.** ***utterance, oracle*** (= III. <span class="bdbLangHebrew">‎מַשָּׂא‎</span>), [[Lam.2.14|Lam 2:14]] ([[BDB#Bu|Bu]] Löhr). 

**3.** ***burden*** [[Zeph.3.18|Zeph 3:18]] (of reproach, figurative). 

**4. a.** ***portion*** (carried to some one), especially from table of king or superior, [[Gen.43.34|Gen 43:34]] (3 t. in verse) ([[BDB#J|J]]; on custom see [[BDB#Di|Di]] and references), [[2Sam.11.8|2Sam 11:8]] so read perhaps also [[2Sam.19.43|2Sam 19:43]] (for [[BDB#MT|ᵑ0]] <span class="bdbLangHebrew">‎נִשֵּׂאת‎</span>) [[BDB#Gr|Gr]] [[BDB#HPS|HPS]]. 
$\quad$ $\quad$ **b.** perhaps also more generally, ***present, largess*** from a superior [[Jer.40.5|Jer 40:5]] ( + <span class="bdbLangHebrew">‎אֲרֻחָה‎</span> ***food-allowance***), [[Esth.2.18|Esth 2:18]] (<span class="bdbLangHebrew">‎כְּיַד הַמֶּלֶךְ‎</span>). 
$\quad$ $\quad$ **c.** exacted or enforced ***gift***, from inferior, <span class="bdbLangHebrew">‎בַּר‎</span> <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎מ‎</span> [[Amos.5.11|Amos 5:11]] (compare Phoenician above; see also  <span class="bdbLangHebrew">‎מנחה‎</span>). 
$\quad$ $\quad$ **d.** sacred ***contribution, tax*** <span class="bdbLangHebrew">‎משֶׁה‎</span> <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎מ‎</span> (<span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎י‎</span>) <span class="bdbLangHebrew">‎עֶבֶד‎</span> [[2Chr.24.6|2Chr 24:6]]; [[2Chr.24.9|2Chr 24:9]]. 
$\quad$ $\quad$ **e.** ***offering*** to <span class="bdbLangHebrew">‎׳‎</span><span class="bdbLangHebrew">‎י‎</span> [[Ezek.20.40|Ezek 20:40]].
