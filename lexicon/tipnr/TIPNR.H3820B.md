Leb-kamai is a cryptic name mentioned in [[Jer.51.1|Jeremiah 51:1]] as part of a prophecy against Babylon. The verse states, "Thus says the LORD: 'Behold, I will raise up against Babylon, against those who dwell in Leb-kamai, a destroying wind.'" The name Leb-kamai is generally understood to be a code or cipher for Chaldea, another name for Babylonia. The exact meaning and reason for using this cryptic name are not explicitly stated in the Bible.
