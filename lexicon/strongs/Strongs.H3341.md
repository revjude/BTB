<big><span class="ashebrew">‎יָצַת‎</span></big> **yâtsath** <small>*(yaw-tsath')*</small> ==**to kindle**==  <sup><small>[[TEHMC#H:V|H:V]]</small></sup> 
<small>a primitive root</small>

$\quad$ burn (up), be desolate, set (on) fire (fire), kindle.
- ***to burn*** or ***set on fire***
-  figuratively, ***to desolate***

**1)** to kindle, burn, set on fire
$\quad$ **1a)** (Qal) to kindle
$\quad$ **1b)** (Niphal)
$\quad$ $\quad$ **1b1)** to be kindled
$\quad$ $\quad$ **1b2)** to be left desolate
$\quad$ **1c)** (Hiphil) to set on fire, kindle
