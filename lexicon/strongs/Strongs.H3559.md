<big><span class="ashebrew">‎כּוּן‎</span></big> **kûwn** <small>*(koon)*</small> ==**to establish: prepare**==  <sup><small>[[TEHMC#H:V|H:V]]</small></sup> 
<small>a primitive root</small>

$\quad$ certain(-ty), confirm, direct, faithfulness, fashion, fasten, firm, be fitted, be fixed, frame, be meet, ordain, order, perfect, (make) preparation, prepare (self), provide, make provision, (be, make) ready, right, set (aright, fast, forth), be stable, (e-) stablish, stand, tarry, [idiom] very deed.
- properly, ***to be erect*** (i.e. stand perpendicular)
-  hence (causatively) to set up, in a great variety of applications, whether literal (establish, fix, prepare, apply), or figurative (appoint, render sure, proper or prosperous)

**1)** to be firm, be stable, be established
$\quad$ **1a)** (Niphal)
$\quad$ $\quad$ **1a1)** to be set up, be established, be fixed
$\quad$ $\quad$ $\quad$ **1a1a)** to be firmly established
$\quad$ $\quad$ $\quad$ **1a1b)** to be established, be stable, be secure, be enduring
$\quad$ $\quad$ $\quad$ **1a1c)** to be fixed, be securely determined
$\quad$ $\quad$ **1a2)** to be directed aright, be fixed aright, be steadfast (moral sense)
$\quad$ $\quad$ **1a3)** to prepare, be ready
$\quad$ $\quad$ **1a4)** to be prepared, be arranged, be settled
$\quad$ **1b)** (Hiphil)
$\quad$ $\quad$ **1b1)** to establish, set up, accomplish, do, make firm
$\quad$ $\quad$ **1b2)** to fix, make ready, prepare, provide, provide for, furnish
$\quad$ $\quad$ **1b3)** to direct toward (moral sense)
$\quad$ $\quad$ **1b4)** to arrange, order
$\quad$ **1c)** (Hophal)
$\quad$ $\quad$ **1c1)** to be established, be fastened
$\quad$ $\quad$ **1c2)** to be prepared, be ready
$\quad$ **1d)** (Polel)
$\quad$ $\quad$ **1d1)** to set up, establish
$\quad$ $\quad$ **1d2)** to constitute, make
$\quad$ $\quad$ **1d3)** to fix
$\quad$ $\quad$ **1d4)** to direct
$\quad$ **1e)** (Pulal) to be established, be prepared
$\quad$ **1f)** (Hithpolel) to be established, be restored
