<big><span class="ashebrew">‎נָסַךְ‎</span></big> **nâçak** <small>*(naw-sak')*</small> ==**to pour**==  <sup><small>[[TEHMC#H:V|H:V]]</small></sup> 
<small>a primitive root</small>

$\quad$ cover, melt, offer, (cause to) pour (out), set (up).
- ***to pour out***, especially ***a libation***, or ***to cast*** (metal)
-  by analogy, ***to anoint a******king***

**1)** to pour out, pour, offer, cast
$\quad$ **1a)** (Qal)
$\quad$ $\quad$ **1a1)** to pour out
$\quad$ $\quad$ **1a2)** to cast metal images
$\quad$ $\quad$ **1a3)** to anoint (a king)
$\quad$ **1b)** (Niphal) to be anointed
$\quad$ **1c)** (Piel) to pour out (as a libation)
$\quad$ **1d)** (Hiphil) to pour out libations
$\quad$ **1e)** (Hophal) to be poured out
**2)** to set, install
$\quad$ **2a)** (Qal) to install
$\quad$ **2b)** (Niphal) to be installed
