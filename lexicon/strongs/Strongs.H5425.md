**<big><span class="ashebrew">‎נָתַר‎</span></big>** **nâthar** <small>*(naw-thar')*</small> ==**to start**==  <sup><small>[[TEHMC#H:V|H:V]]</small></sup> 
<small>a primitive root</small>

$\quad$ drive asunder, leap, (let) loose, [idiom] make, move, undo.
- ***to jump***, i.e. ***be violently agitated***
-  causatively, ***to terrify***, ***shake off***, ***untie***

**1)** to start up, tremble, shake, spring up
$\quad$ **1a)** (Qal) to start up
$\quad$ **1b)** (Piel) to leap
$\quad$ **1c)** (Hiphil) to cause to start up
**2)** to loose, let loose, undo, be free, be loose
$\quad$ **2a)** (Hiphil)
$\quad$ $\quad$ **2a1)** to unfasten, loosen
$\quad$ $\quad$ **2a2)** to set free, unbind
