**<big><span class="ashebrew">‎שָׂגַב‎</span></big>** **sâgab** <small>*(saw-gab')*</small> ==**to exalt**==  <sup><small>[[TEHMC#H:V|H:V]]</small></sup> 
<small>a primitive root</small>

$\quad$ defend, exalt, be excellent, (be, set on) high, lofty, be safe, set up (on high), be too strong.
- ***to be*** (causatively, make) ***lofty***, especially ***inaccessible***
-  by implication, ***safe***, ***strong***
-  used literally and figuratively

**1)** to be high, be inaccessibly high
$\quad$ **1a)** (Qal)
$\quad$ $\quad$ **1a1)** to be (too) high (for capture)
$\quad$ $\quad$ **1a2)** to be high (of prosperity)
$\quad$ **1b)** (Niphal)
$\quad$ $\quad$ **1b1)** to be high
$\quad$ $\quad$ **1b2)** to be set on high, be (safely) set on high
$\quad$ $\quad$ **1b3)** to be exalted (of God)
$\quad$ **1c)** (Piel)
$\quad$ $\quad$ **1c1)** to set on high, set (securely) on high
$\quad$ $\quad$ **1c2)** to exalt, exalt (in effective hostility)
$\quad$ **1d)** (Pual) to be set (securely) on high
$\quad$ **1e)** (Hiphil) to act exaltedly
